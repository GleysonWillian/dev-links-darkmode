# DevLinks Dark Mode 🌙

Projeto de página de links estilo "link na bio", desenvolvido para centralizar meus projetos e informações como desenvolvedor.

## 📌 Descrição
Projeto desenvolvido com base no curso Discover da Rocketseat.

Além do conteúdo porposto no curso, implementei melhorias próprias, como:

- Adição de projetos pessoais na página
- Implementação do modo escuro (dark mode)
- Personalização de layout e estilos

## 🚀 Funcionalidades

- Alternância entre tema claro e escuro
- Layout responsivo
- Links personalizados
- Interface simples e moderna

## 🛠 Tecnologias

- HTML
- CSS
- JavaScript

## 💡 Aprendizados:
- Manipulação do DOM com JavaScript
- Uso de classes para alternar temas
- Organização de layout com CSS
- Responsividade
